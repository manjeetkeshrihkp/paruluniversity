Chapter 6: Pointers, Dynamic Memory Allocation & File Management

Memory Handling • Runtime Allocation • Permanent Data Storage in C

Basics of Pointers

A pointer is a variable that stores the memory address of another variable. Pointers allow direct memory access and efficient manipulation of data.

Address Storage
Efficient Access
Low-level Control
int x = 10;
int *p = &x;

printf("%d", *p);   // value
printf("%p", p);    // address

Pointer to Pointer

A pointer to pointer stores the address of another pointer. It is useful in dynamic memory allocation and complex data structures. 

int a = 5;
int *p = &a;
int **q = &p;

printf("%d", **q);

Pointer and Array Relationship

Array name acts as a constant pointer to its first element. Pointer arithmetic allows traversal of arrays efficiently. 

int arr[5] = {1,2,3,4,5};
int *p = arr;

for(int i=0;i<5;i++)
  printf("%d ", *(p+i));

Pointer to Array

A pointer to array points to the entire array rather than a single element. 

int arr[5] = {10,20,30,40,50};
int (*p)[5] = &arr;

printf("%d", (*p)[2]);

Array of Pointers

An array of pointers stores multiple addresses. It is commonly used to handle strings efficiently. 

char *names[] = {"C", "C++", "Python"};

for(int i=0;i<3;i++)
  printf("%s\n", names[i]);

Functions Returning Pointers

Functions can return pointers, but the returned address must remain valid after the function execution. 

int* getValue() {
  static int x = 100;
  return &x;
}
Never return address of a local (stack) variable

Introduction to Dynamic Memory Allocation

Dynamic memory allocation allows memory to be allocated at runtime using heap memory, making programs flexible and efficient.

malloc()
calloc()
realloc()
free()

malloc()

Allocates a single block of memory with uninitialized values. 

int *arr = (int*)malloc(5 * sizeof(int));
if(arr == NULL) return 1;

calloc()

Allocates multiple blocks and initializes all values to zero. 

int *arr = (int*)calloc(5, sizeof(int));

realloc()

Resizes previously allocated memory without losing existing data. 

arr = (int*)realloc(arr, 10 * sizeof(int));

free()

Releases dynamically allocated memory back to the system. 

free(arr);

Introduction to File Management in C

File handling allows permanent storage of data. C supports both text and binary file operations.

Standard File Handling Functions

C provides built-in functions to create, read, write and close files.

fopen()
fclose()
fprintf()
fscanf()
fgetc()
fputc()
FILE *fp = fopen("data.txt","w");
fprintf(fp,"Hello PPS");
fclose(fp);

Summary

This chapter explains how memory and files are handled in C. Pointers and dynamic memory allocation provide flexibility, while file handling enables permanent data storage.
