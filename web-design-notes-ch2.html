Chapter 2: HTML Elements & Forms

In this chapter, we study lists, tables, forms, and semantic HTML elements.

1. Lists & Tables

HTML Lists

Lists are used to display information in an organized manner.

Unordered List

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

Ordered List

<ol>
  <li>Frontend</li>
  <li>Backend</li>
  <li>Database</li>
</ol>

Description List

<dl>
  <dt>HTML</dt>
  <dd>Markup Language</dd>
</dl>

HTML Tables

Tables are used to display data in rows and columns.

<table>
  <tr>
    <th>Name</th>
    <th>Course</th>
  </tr>
  <tr>
    <td>Amit</td>
    <td>BTech</td>
  </tr>
</table>
Tag Description
<table>Defines table
<tr>Table row
<th>Table header
<td>Table data

2. Forms

Forms are used to collect user input and send data to a server.

Basic Form Structure

<form>
  <label>Name:</label>
  <input type="text">
</form>

Common Input Types

  • text
  • password
  • email
  • number
  • radio
  • checkbox

Form Example

<form>
  <label>Email:</label>
  <input type="email">

  <label>Gender:</label>
  <input type="radio"> Male
  <input type="radio"> Female

  <button>Submit</button>
</form>

3. Semantic Tags

Semantic tags clearly describe the meaning of content. They improve readability, accessibility, and SEO.

Tag Purpose
<header>Header section
<nav>Navigation
<section>Section of page
<article>Independent content
<aside>Side content
<footer>Footer

Semantic tags are better than using only <div> and <span>.

Summary

  • Lists organize information clearly
  • Tables display data systematically
  • Forms collect user input
  • Semantic tags improve structure and SEO
