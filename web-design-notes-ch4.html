Chapter 4: Responsive & Advanced CSS

This chapter explains how to make websites responsive and interactive using media queries, transitions, and animations.

1. Media Queries

Media queries allow websites to adapt their layout based on screen size. They are essential for responsive web design.

Why Media Queries are Used

  • Support mobile, tablet, and desktop screens
  • Improve user experience
  • Make websites responsive

Syntax of Media Query

@media (max-width: 768px) {
  body {
    background-color: lightblue;
  }
}

The above code applies styles when the screen width is 768px or smaller.

2. CSS Transitions

Transitions allow smooth change between CSS property values. They improve the visual experience of a website.

Transition Properties

  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay

Transition Example

button {
  background: blue;
  transition: background 0.3s ease;
}

button:hover {
  background: red;
}

3. CSS Animations

CSS animations allow elements to change style over time using keyframes. They are more powerful than transitions.

Key Animation Properties

  • @keyframes
  • animation-name
  • animation-duration
  • animation-iteration-count

Animation Example

@keyframes move {
  from { left: 0; }
  to { left: 100px; }
}

.box {
  position: relative;
  animation: move 2s infinite;
}

Summary

  • Media queries make websites responsive
  • Transitions create smooth effects
  • Animations add movement to web pages
