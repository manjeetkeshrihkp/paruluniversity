Chapter 3: CSS Basics & Layout

This chapter explains how CSS controls layout and spacing using the box model, flexbox, and grid.

1. CSS Box Model

The CSS Box Model defines how elements are structured and spaced on a webpage. Every HTML element is treated as a rectangular box.

Components of Box Model

  • Content – Actual text or image
  • Padding – Space between content and border
  • Border – Surrounds the padding
  • Margin – Space outside the border

Box Model Example

div {
  width: 200px;
  padding: 20px;
  border: 2px solid black;
  margin: 10px;
}

Total width = width + padding + border + margin

2. CSS Flexbox

Flexbox (Flexible Box Layout) is used to design one-dimensional layouts. It makes alignment and spacing easier.

Flex Container

.container {
  display: flex;
}

Common Flexbox Properties

  • flex-direction – Row or column
  • justify-content – Horizontal alignment
  • align-items – Vertical alignment
  • flex-wrap – Wrap items

Flexbox Example

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

3. CSS Grid

CSS Grid is used to create two-dimensional layouts (rows and columns). It is ideal for complex layouts.

Grid Container

.grid {
  display: grid;
}

Grid Properties

  • grid-template-columns
  • grid-template-rows
  • gap

Grid Example

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

Summary

  • Box Model controls spacing and size
  • Flexbox is best for one-dimensional layouts
  • Grid is best for two-dimensional layouts
