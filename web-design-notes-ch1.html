Chapter 1: Introduction to HTML & CSS

This chapter introduces the fundamentals of HTML and CSS used in web development.

1. HTML Basics

HTML stands for HyperText Markup Language. It is used to create the structure of web pages. HTML is not a programming language, it is a markup language.

Features of HTML

  • Easy to learn and use
  • Platform independent
  • Supported by all web browsers
  • Works with CSS and JavaScript
  • Supports multimedia content

Basic HTML Structure

<!DOCTYPE html>
<html>
<head>
  <title>My Web Page</title>
</head>
<body>
  <h1>Hello World</h1>
  <p>Welcome to my website</p>
</body>
</html>

Common HTML Tags

TagDescription
<h1> – <h6>Headings
<p>Paragraph
<a>Hyperlink
<img>Image
<div>Container

HTML Attributes

<a href="https://google.com">Google</a>

2. CSS Introduction

CSS stands for Cascading Style Sheets. CSS is used to style and design HTML elements.

Why CSS is Used

  • Improves website appearance
  • Separates content from design
  • Creates responsive layouts
  • Reduces code repetition

CSS Syntax

selector {
  property: value;
}

Example

p {
  color: blue;
  font-size: 16px;
}

3. Types of CSS

1. Inline CSS

CSS applied directly inside an HTML tag.

<p style="color:red;">Hello</p>

2. Internal CSS

CSS written inside the <style> tag in the head section.

<style>
p { color: blue; }
</style>

3. External CSS (Recommended)

CSS written in a separate file and linked to HTML.

<link rel="stylesheet" href="style.css">
TypeReusableBest Practice

Summary

  • HTML provides structure to webpages
  • CSS styles and designs webpages
  • External CSS is the most professional approach
