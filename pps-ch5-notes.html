Chapter 5: User-Defined Functions, Structures & Unions

Modular Programming • Structured Data • Memory Optimization

User-Defined Functions

Definition: A user-defined function is a block of code written by the programmer to perform a specific task and can be reused multiple times.

User-defined functions help divide a large program into smaller modules, improving readability and maintainability. 

int add(int a, int b) {
    return a + b;
}

Function Prototype

Definition: A function prototype is a declaration that informs the compiler about a function’s name, return type and parameters before its use. 

int add(int, int);

Function Definition & Calling

Definition: Function definition contains the actual code of the function, and function calling transfers control to that function. 

int main() {
    int sum = add(5, 10);
    return 0;
}

Parameters & Parameter Passing

Definition: Parameters are variables used to pass data to a function. Parameter passing decides how data is transferred between functions. 

void change(int x) { x = 20; }
void update(int *x) { *x = 20; }

Recursive Functions

Definition: A recursive function is a function that calls itself until a base condition is satisfied. 

int fact(int n) {
    if(n == 0) return 1;
    return n * fact(n - 1);
}

Macros

Definition: A macro is a preprocessor directive that replaces a piece of code or value before compilation. 

#define PI 3.14159
#define SQR(x) ((x)*(x))

Pre-Processing Directives

Definition: Pre-processing directives are commands executed by the compiler before actual compilation begins.

#include
#define
#ifdef
#ifndef

Structures

Definition: A structure is a user-defined data type that groups variables of different data types under a single name. 

struct Student {
    int roll;
    float marks;
};

Declaring & Initializing Structures

Definition: Structure variables are declared using the structure name and initialized with values. 

struct Student s1 = {101, 88.5};

Accessing Structure Members

Definition: Structure members are accessed using the dot (.) operator. 

printf("%d", s1.roll);

Copying & Comparison of Structures

Definition: Copying means assigning one structure to another, while comparison must be done member-wise. 

struct Student s2 = s1;

Arrays of Structures

Definition: An array of structures stores multiple records of the same structure type. 

struct Student class[5];

Structures within Structures

Definition: A structure can contain another structure as its member. 

struct Address {
    char city[20];
};

struct Student {
    int roll;
    struct Address addr;
};

Structures and Functions

Definition: Structures can be passed to functions by value or by reference. 

void display(struct Student s);
void update(struct Student *s);

Unions

Definition: A union is a user-defined data type in which all members share the same memory location. 

union Data {
    int i;
    float f;
};

Summary

This chapter explains modular programming using user-defined functions and efficient data handling using structures and unions, which are essential for real-world programming.
