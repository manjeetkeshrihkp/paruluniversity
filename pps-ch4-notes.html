4Ô∏è‚É£ Parameters & Parameter Passing

Parameters are used to pass data from one function to another. There are two main methods of parameter passing in C.

üîπ Pass by Value

A copy of data is passed. Changes do NOT affect the original variable. 

void swap(int a, int b) {
    int t = a;
    a = b;
    b = t;
}
üîπ Pass by Reference

Address of data is passed. Changes DO affect the original variable. 

void swap(int *a, int *b) {
    int t = *a;
    *a = *b;
    *b = t;
}

5Ô∏è‚É£ Passing Arrays to Functions

        arr[i] += 10;
}
