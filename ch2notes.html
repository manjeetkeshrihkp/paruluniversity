Chapter 2: Multivariate Calculus

Functions of Several Variables, Partial Derivatives, Jacobian, Taylor Series & Optimization

1. Functions of Several Variables

A multivariable function is a function that depends on two or more independent variables.

z = f(x, y),   z = f(x, y, z)

Geometrically, a function z = f(x, y) represents a surface in three-dimensional space.

Level Curves

Level curves are curves obtained by setting f(x, y) = c, where c is constant.

Level curves help visualize surfaces on a 2D plane.

2. Partial Derivatives

Partial derivatives measure the rate of change of a function with respect to one variable while keeping other variables constant.

Definition

∂f/∂x = lim(h→0) [f(x+h, y) − f(x, y)] / h
∂f/∂y = lim(h→0) [f(x, y+h) − f(x, y)] / h

Higher Order Partial Derivatives

∂²f/∂x² , ∂²f/∂y² , ∂²f/∂x∂y
If second-order partial derivatives are continuous, then ∂²f/∂x∂y = ∂²f/∂y∂x.

3. Chain Rule

The chain rule is used when variables are functions of other variables.

Two Variable Form

dz/dt = (∂z/∂x)(dx/dt) + (∂z/∂y)(dy/dt)
Useful in physics, machine learning gradients and optimization problems.

4. Jacobian

The Jacobian is a determinant used for transformation of variables.

Jacobian of (u, v) with respect to (x, y)

J = ∂(u, v)/∂(x, y) = | ∂u/∂x ∂u/∂y ; ∂v/∂x ∂v/∂y |
Jacobian plays a key role in multiple integrals and coordinate transformations.

5. Taylor Expansion

Taylor expansion approximates a function near a given point.

Two Variable Taylor Series

f(x,y) ≈ f(a,b) + (x−a)fₓ + (y−b)fᵧ + 1/2[(x−a)²fₓₓ + 2(x−a)(y−b)fₓᵧ + (y−b)²fᵧᵧ]
Used in numerical methods, optimization and AI models.

6. Maxima and Minima

Critical points occur where first-order partial derivatives vanish.

fₓ = 0 , fᵧ = 0

Second Derivative Test

D = fₓₓ fᵧᵧ − (fₓᵧ)²
  • If D > 0 and fₓₓ > 0 → Minimum
  • If D > 0 and fₓₓ < 0 → Maximum
  • If D < 0 → Saddle point

7. Lagrange Multipliers

Lagrange multipliers are used to find extrema of a function subject to constraints.

Method

∇f = λ ∇g

Where g(x, y) = 0 is the constraint.

Extremely important for optimization in economics, engineering and AI.
