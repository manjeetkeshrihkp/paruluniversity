Chapter 4: Vector Calculus

Vector Functions, Differential Operators, Line & Surface Integrals, Green, Gauss and Stokes Theorems

1. Vector Functions

A vector function assigns a vector to each value of a scalar parameter. It is commonly used to describe curves and motion in space.

r(t) = x(t)i + y(t)j + z(t)k

Here, x(t), y(t), z(t) are scalar functions of parameter t.

Derivative of Vector Function

dr/dt = dx/dt i + dy/dt j + dz/dt k
The derivative represents the velocity vector in physics.

2. Gradient (∇f)

The gradient of a scalar function points in the direction of the maximum rate of increase of the function.

∇f = (∂f/∂x)i + (∂f/∂y)j + (∂f/∂z)k

Geometrical Interpretation

The gradient is perpendicular (normal) to the level surface f(x, y, z) = constant.

Used to find normal vectors and direction of steepest ascent.

3. Divergence (∇·F)

Divergence measures the outflow or inflow of a vector field from a point.

∇·F = ∂F₁/∂x + ∂F₂/∂y + ∂F₃/∂z

If divergence is positive, the field behaves like a source; if negative, like a sink.

Widely used in fluid flow and electromagnetism.

4. Curl (∇×F)

Curl measures the rotational tendency of a vector field.

∇×F = | i j k ; ∂/∂x ∂/∂y ∂/∂z ; F₁ F₂ F₃ |

If curl is zero, the field is called irrotational.

Curl is crucial in understanding circulation and magnetic fields.

5. Line Integrals

Line integrals calculate the integral of a vector field along a curve.

∫C F · dr

Used to compute work done by a force along a path.

Path Independence

If curl F = 0, the line integral is path independent.

6. Surface Integrals

Surface integrals extend integration to curved surfaces.

∬S F · n dS

This represents the flux of vector field across a surface.

Used in heat flow and fluid mechanics.

7. Green’s Theorem

Green’s theorem relates a line integral around a closed curve to a double integral over the enclosed region.

∮ (P dx + Q dy) = ∬ (∂Q/∂x − ∂P/∂y) dA
Applicable only in 2D for simple closed curves.

8. Gauss Divergence Theorem

Gauss theorem converts a surface integral into a volume integral.

∬S F · n dS = ∭V (∇·F) dV

This theorem simplifies flux calculations greatly.

9. Stokes’ Theorem

Stokes theorem relates a surface integral of curl to a line integral along the boundary curve.

∮ F · dr = ∬ (∇×F) · n dS
A powerful theorem connecting circulation and rotation.

10. Applications of Vector Calculus

  • Fluid flow and continuity equations
  • Electromagnetic field analysis
  • Heat transfer and diffusion
  • Engineering and AI-based simulations
Vector calculus provides a unified framework to study physical systems.
